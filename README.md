# Farvig
Module eindopdracht Java

Ik wil een programma maken waarin een klant een zogenaamde “Kaart op maat” kan maken.
In mijn webshop Farvig bied ik de optie aan dat ik op basis van de wensen van de klant een kaart  ontwerp met een persoonlijke tekst.
Een versimpelde versie wil ik dus maken met behulp van Java waarbij de klant zelf in staat is een kaartformaat te kiezen, de achtergrondkleur, het lettertype en de tekst kan invoeren.

Ik maak: een Main Classe genaamd “Main”. Daarin staat de main method.
Deze moet straks voor mij het hele programma in werking zetten.

Daarnaast wil ik de volgende classes aanmaken:
-	Kaart op maat
-	Tekst
-	Achtergrondkleur
-	Lettertype
-	Kaartformaat

Het  werkt nu, maar ik wil nog  graag ervoor zorgen dat alle gegeven antwoorden van de klant in 1 overzicht komen.  Nog geen idee hoe ...
